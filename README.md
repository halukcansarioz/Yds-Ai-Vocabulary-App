# 🎓 YDS AI Vocabulary App

A modern vocabulary learning application developed for YDS and YÖKDİL exam preparation.

## 🎯 Target Audience

This project is designed for:

- YDS students
- YÖKDİL students
- Academic English learners
- Vocabulary-focused language learners

## ✨ Features

- 📚 Vocabulary Management
  - Add words
  - Edit words
  - Delete words
  - Search words

- 🎴 Flashcard Study Mode
  - Randomized flashcards
  - Instant review system
  - Progress tracking

- 🧠 Spaced Repetition System (SRS)
  - Daily reviews
  - Weak word detection
  - Knowledge level tracking

- 📊 Analytics Dashboard
  - Total vocabulary count
  - Known / Medium / Weak statistics
  - Success rate
  - Total review count
  - Most reviewed words

- 📥 CSV Import
  - Bulk vocabulary loading
  - Easy dataset expansion

## 🚀 Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
